# 策略公开方案

## 目标

公开这套策略的研究框架，但不公开任何可以直接复现 live 交易的钱包、secret 或生产配置。

## 推荐口径

推荐对外口径：

- 这是一套 bond 风格的 Polymarket 研究框架
- 它在后期曾出现过一段稳定盈利窗口
- 它后来没有继续沿原路线扩张，而是切换到了别的方向
- `5/15min` 方向也曾出现过短暂正收益窗口，但工程难度和竞争强度太高，因此停止继续开发

## 不公开的内容

- 私钥
- API credentials
- live compose
- production state
- 当前新算法细节

## 公开的重点

- 方法论
- 研究边界
- 市场结构理解
- 为什么这套思路能成立
- 为什么最终选择切换方向
