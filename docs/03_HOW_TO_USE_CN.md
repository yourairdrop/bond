# 这套策略该怎么使用

这里说的“使用”，不是指把 live 配置抄过去直接开真仓。

更准确地说，这套策略现在适合这样用：

## 1. 当成研究框架用

最适合公开的部分，不是 live 执行细节，而是研究框架：

- 如何选市场
- 如何定义 `p_fair`
- 如何做 dry-run
- 如何看 MTM
- 如何判断一个策略只是短期赚钱，还是结构上真的有 edge

## 2. 不要把天气 exact-temperature 直接当成可直接上线的模块

旧 weather 策略后面已经证明：

- exact-temperature 市场不能只靠粗粒度 `p_fair` 去做
- 高价、短期限、流动性尚可，不等于有真实 edge

因此，更合理的用法不是直接恢复旧天气 live，而是把它当作 replay、分层和模型重建的样本框架。

## 3. `5/15min` 方向只适合拿来理解，不适合直接跟做

bond 思路在 `5/15min` 市场里，确实曾短暂跑出过正收益窗口，但这类市场现在更像是高竞争、高速度、高细节的执行战场。

所以它更适合用来理解：

- fee 如何吃掉 edge
- queue position 为什么重要
- 为什么看起来有 alpha 的短周期策略，最后不一定值得继续投工程成本

## 4. 如果要复现，推荐顺序

1. 先跑 dry-run
2. 再做 replay 和坏样本分析
3. 最后才谈 shadow，更不要直接跳到 live
