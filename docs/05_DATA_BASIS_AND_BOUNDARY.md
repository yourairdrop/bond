# 数据基础与口径边界

## 这份说明的作用

公开策略最容易出问题的地方，不是文案写得不够好，而是：

- 赚钱窗口和长期可行被混为一谈
- paper、dry-run、live 被混为一谈
- 老数据和当前可行被混为一谈

所以这里单独把边界写清楚。

## 可公开口径

推荐使用：

- `后期曾出现过一段稳定盈利窗口`
- `短暂跑出过 3 天正收益的 paper / research 窗口`
- `后来主动停掉，因为模型或市场结构不再适合继续投入`

不推荐使用：

- `已经长期稳定盈利`
- `这是一套可复制的稳定套利系统`
- `5/15min 市场已经跑通`

## 最终定位

这套公开材料最真实的定位是：

> 一套真实跑过、真实出现过阶段性盈利窗口、最后又主动停止并切换方向的策略研究框架。

这个定位比“神化它”更可靠，也更适合长期公开使用。
